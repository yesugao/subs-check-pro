保存方法配置

选择保存方法时，请更改 save-method 配置。

  • 本地保存：保存到 ./output/sub 文件夹。
  • R2：保存到 Cloudflare R2（参见 R2 保存方法）。
  • Gist：保存到 GitHub Gist（参见 Gist 保存方法）。
  • WebDAV：保存到 WebDAV 服务器（参见 WebDAV 保存方法）。
  • S3：保存到 S3 对象存储。
